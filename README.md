# 스프링 부트와 AWS로 혼자 구현하는 웹 서비스 - 저자 이동욱
- 이 프로젝트는 저자 이동욱님의 책 '스프링 부트와 AWS로 혼자 구현하는 웹 서비스' 를 기반으로 공부한 내용입니다.
- 이 프로젝트를 진행했던 날 기준으로 해당 책을 공부하였기 때문에 Spring boot, java, gradle version 에 따른 적용 방법이 상이할 수 있습니다.
- 중지되었거나 사용하지 않는 버전, 기술들에 있어서 대체한 부분이 있기에 그 점을 참고해주시기 바랍니다.

### 사용기술
- Spring Boot 3.4.3
- Java 21
- Gradle 8.13
- JUnit5
- Spring Security + OAuth2.0
- Thymelef
